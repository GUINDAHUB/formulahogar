# Logos Directory

Coloca tus logos aquí sin necesidad de renombrarlos con nombres específicos.

El sistema auto-detecta automáticamente las imágenes usando estos patrones:
- `logo1.png`, `logo2.png`, etc. (hasta logo20)
- `partner1.png`, `client1.png`, `sponsor1.png`, `brand1.png`
- Formatos soportados: PNG, JPG, JPEG, SVG, WEBP

**Recomendaciones:**
- Formato: PNG con fondo transparente (preferido)
- Alto: 100-150px
- Peso: < 50KB por logo optimizado
- El slider se ajusta automáticamente al número de logos que añadas
